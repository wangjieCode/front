# 前端设计风格指南

## 1. 核心视觉系统

### 1.1 色彩体系
项目采用统一的紫色系作为品牌主色调，体现现代感与科技感。

*   **主色调 (Primary)**: `#7c5cff` (Ant Design 变量名: `colorPrimary`)
    *   用于关键按钮、激活状态图标、选中高亮等。
*   **悬停色 (Hover)**: `#6b4ce0`
    *   用于交互元素的悬停状态。
*   **点击色 (Active)**: `#5a3dcf`
    *   用于交互元素的点击/按下状态。
*   **阴影 (Shadow)**: `rgba(124, 92, 255, 0.3)`
    *   用于主按钮和重要卡片的投影，增加层次感。

### 1.2 布局结构
采用经典的左右分栏布局：

*   **侧边栏 (Sidebar)**:
    *   固定宽度 `300px`，浅色背景 (`#ffffff`)。
    *   `position: fixed`，高度 `100vh`。
    *   右侧有 `1px` 的边框 (`#f0f0f0`) 区分内容区。
*   **主要内容区 (Main Content)**:
    *   浅灰色背景 (`#f5f5f5`)。
    *   左侧预留 `300px` 边距。
    *   包含顶部导航栏 (Header) 和内容视窗 (Content)。

### 1.3 视觉风格细节
*   **圆角 (Border Radius)**:
    *   标准圆角: `8px` (按钮、列表项)。
    *   大圆角: `12px` (卡片、输入框)。
    *   超大圆角: `16px` - `24px` (聊天气泡、部分操作按钮)。
*   **阴影 (Box Shadow)**:
    *   注重柔和的扩散阴影，避免生硬的黑色边框。
    *   示例: `0 4px 12px rgba(0, 0, 0, 0.1)` (卡片悬停)。
*   **渐变 (Gradients)**:
    *   在主要行动点（如发送按钮、用户消息气泡）使用线性渐变。
    *   `linear-gradient(135deg, #7c5cff 0%, #6b4ce0 100%)`。

## 2. 组件设计规范

### 2.1 按钮 (Buttons)
*   **Primary 按钮**: 使用品牌紫色渐变背景，带有彩色投影，悬停时会有轻微的上浮动画 (`transform: translateY(-1px)`)。
*   **图标按钮**: 圆形或圆角方形，色彩与文字颜色保持一致。

### 2.2 卡片 (Cards)
*   背景色: `#ffffff`。
*   边框: `1px solid #f0f0f0`。
*   交互: 悬停时产生轻微上浮和阴影加深效果。

### 2.3 聊天气泡 (Chat Bubbles)
*   **用户**: 紫色渐变背景，白色文字，右上角圆角特殊处理。
*   **AI**: 白色背景，深色文字，左上角圆角特殊处理，带有灰色边框。

### 2.4 其他元素
*   **Tag/Badge**: 使用淡紫色背景 (`rgba(124, 92, 255, 0.1)`) 搭配主色文字。
*   **滚动条**: 细长 (`6px`)，颜色为浅灰 (`#d9d9d9`)，圆角处理。

## 3. 实现指南
在开发新页面时，请遵循以下 CSS 变量和 React 组件规范：

*   使用 `App.css` 中定义的 CSS 变量（如 `var(--primary-color)`）。
*   尽量复用 `ConversationView` 和 `ProjectsPage` 中的卡片和布局模式。
*   保持交互一致性（Hover、Active 状态）。
