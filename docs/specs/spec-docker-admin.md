# 规格：Docker 管理

## 基本信息

- 名称：Docker 管理接口
- 负责人：未指定
- 创建日期：2026-01-30
- 最近更新：2026-01-30

## 背景

- 需要远程管理容器与镜像。

## 目标

- 提供基础的 Docker 容器与镜像操作。

## 非目标

- 不提供多租户隔离。

## 范围

- In：容器、镜像、构建、日志、统计。
- Out：K8s 管理。

## 业务规则

- Docker 连接通过环境变量配置。

## 需求

### 功能需求

- F1：容器列表与控制。
- F2：镜像拉取与构建。
- F3：创建容器并启动。

### 非功能需求

- N1：错误需可定位。

## 用户体验

- API 统一返回 success 与 error。

## 数据与接口

- /api/docker/*

## 验收标准

- A1：容器列表可返回数据。

## 风险与依赖

- 依赖远程 Docker API 与 SSH 配置。
