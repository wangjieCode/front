# 规格：代码变更记录

## 背景

- 需要精确追踪 AI 每次修改的代码。

## 目标

- 每条 AI 消息记录变更列表。

## 非目标

- 不提供对话级别汇总表。

## 范围

- In：message_metadata.code_changes。
- Out：前端展示细节。

## 业务规则

- 只有 AI 消息会写入 code_changes。

## 需求

### 功能需求

- F1：解析变更并保存到消息元数据。
- F2：支持按消息读取变更。

### 非功能需求

- N1：无变更时允许为空。

## 用户体验

- 变更可用于审计与展示。

## 数据与接口

- message_metadata.code_changes。

## 验收标准

- A1：AI 消息保存后可查询变更列表。
