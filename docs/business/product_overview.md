# 产品概览

## 产品名称

- `代码伙计 / Code Mate`

## Slogan

- `少折腾，多交付。/ Less hassle, more delivery.`

## 核心价值

- 用对话驱动代码修改与交付
- 以会话为单位隔离工作区，降低冲突
- 统一记录代码变更与操作轨迹

## 主要能力

- 项目管理：登记仓库、维护工作目录
- 会话对话：自然语言驱动改动
- 代码变更记录：每条 AI 消息可追溯
- 交付：推送分支与手动创建 MR
- 预览：为会话工作区启动可访问服务

## 目标用户

- 需要快速改动并验证代码的开发人员
- 需要管理多个项目与会话的团队成员
